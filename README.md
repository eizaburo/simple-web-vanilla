# simple-web-vanilla

シンプルなWebUIをHTML、CSSおよび素のJavaScript（Vanilla JS）で実装したもの。

## ブランチ情報

- main
  - シンプルなAPIとの連携を想定した実装

## 前提

- 環境
  - macOS
    - 特に依存要素は無いが、作成環境がmacOS
  - Chrome
    - ブラウザ非依存ですが、確認はChromeのみで実施
  - Node.jsがインストール済み
    - 実装には関係ないが、動作確認に利用


## 利用方法

クライアント技術のみで構築されているので、リポジトリをcloneしてHTMLファイルを開くだけです。

```bash
git clone https://github.com/eizaburo/simple-web-vanilla.git
cd simple-web-vanilla
npx serve .
```

もしくは、

```bash
open index.html
```

>ダブルクリックでindex.htmlを開いてもOKです。
